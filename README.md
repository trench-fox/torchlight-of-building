# Torchlight of Building

Torchlight Infinite character build planner and dps calculator.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- TanStack Start + Vite
- React 19
- TypeScript
- Tailwind CSS 4
- Zustand (state management)
